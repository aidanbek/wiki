OpenAPI Specification (formerly Swagger): standard для describing REST APIs. YAML/JSON format. Describes: endpoints, methods, parameters, request/response schemas, auth. Machine-readable (code generation, validation, mocking). Tools: Swagger UI (interactive docs), Swagger Editor, code generators. API-first design: define contract before implementation. Versioning: OpenAPI 2.0 (Swagger) → 3.0 → 3.1 (JSON Schema alignment).
