# Система типов

## Список базовых типов

Типы с 1 по 4 - скалярные типы (простые, неделимые)

1. Целые числа integer: Целые числа без десятичных знаков, например 42, -10, 0.
2. Числа с плавающей точкой float: Числа с десятичными знаками, например 3.14, -2.5, 0.0.
3. Строки string: Последовательности символов, заключенные в одинарные или двойные кавычки, например 'Hello', "World".
4. Булев тип boolean: Логическое значение true или false.
5. Массивы array: Упорядоченные коллекции элементов, каждый из которых имеет свой индекс или ключ.
6. Объекты object: Экземпляры классов, содержащие свойства и методы.
7. Ресурсы resource: Ссылки на внешние ресурсы, такие как файлы или соединения с базами данных.
8. Null: Особое значение null, которое означает отсутствие значения.
9. Callback/Callable: Замыкания на функции.
10. Псевдотипы pseudo-types: Некоторые псевдотипы, такие как mixed, number, void, iterable, never которые представляют
    собой специальные категории типов данных или ограничения.
