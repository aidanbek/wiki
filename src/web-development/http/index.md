HyperText Transfer Protocol - application layer protocol. Request: method + URL + headers + body. Response: status
code + headers + body. Methods: GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS. Status codes: 1xx (informational), 2xx (
success), 3xx (redirect), 4xx (client error), 5xx (server error). Headers: metadata (Content-Type, Authorization,
Cache-Control). HTTP/1.1: persistent connections, chunked transfer. HTTP/2: multiplexing, server push, header
compression. HTTP/3: QUIC (UDP-based), faster connection establishment.
