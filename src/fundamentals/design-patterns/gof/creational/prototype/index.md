Создание объектов клонированием прототипа вместо конструирования - копирование вместо создания с нуля. Полезно когда
создание дорого (БД запрос, сложная инициализация). clone() метод возвращает копию, shallow vs deep copy. JavaScript
прототипное наследование основано на этом. Динамическая конфигурация через клонирование и модификацию. Избегает жёсткой
привязки к конкретным классам.
