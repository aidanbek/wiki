State

Изменение поведения объекта при изменении внутреннего состояния - объект выглядит меняющим класс. Вместо switch/if по
состоянию, каждое состояние - отдельный класс. Context делегирует State объекту, State инкапсулирует поведение для этого
состояния. Переходы между состояниями явные (State меняет Context.state или Context управляет). Устраняет conditional
logic, делает состояния explicit, легко добавлять новые.
