Суррогат контролирующий доступ к другому объекту - placeholder с тем же интерфейсом, добавляющий контроль. Virtual
Proxy (ленивая инициализация), Protection Proxy (контроль доступа), Remote Proxy (удалённый объект), Smart Proxy (
дополнительная логика). ImageProxy загружает тяжёлое изображение только при обращении. Клиент не знает о proxy, работает
через общий интерфейс. Контроль без изменения реального объекта.
