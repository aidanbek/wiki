Конвертирует интерфейс класса в интерфейс, ожидаемый клиентом - обёртка для несовместимых интерфейсов. Legacy код с
методом oldCalculate() оборачивается адаптером с интерфейсом ICalculator.compute(). Два варианта: object adapter (
композиция) и class adapter (множественное наследование). Позволяет работать с третьесторонними библиотеками, legacy
кодом, несовместимыми системами без их изменения. Wrapper для несовместимости.
