Композиция объектов в древовидные структуры для представления иерархий part-whole, единообразная работа с одиночными
объектами и композициями. FileSystem содержит File и Directory, Directory содержит другие File/Directory - одинаковый
интерфейс. Client работает с Component интерфейсом, не различая Leaf и Composite. Рекурсивная композиция, операции
автоматически распространяются по дереву.
