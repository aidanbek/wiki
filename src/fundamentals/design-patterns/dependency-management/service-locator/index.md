Глобальный реестр сервисов, предоставляющий API для получения зависимостей: var repo =
ServiceLocator.Get<IRepository>(). Централизованная точка доступа к зависимостям, скрывающая их создание и конфигурацию.
Считается антипаттерном - скрытые зависимости (не видны в сигнатуре), затрудняет тестирование (нужно мокать весь
locator), нарушает явность. Работает как Singleton на стероидах. Используется в legacy code или когда DI невозможна (
static contexts).
