фMutex (Mutual Exclusion)

Блокировка обеспечивающая эксклюзивный доступ к ресурсу - только один поток может владеть mutex одновременно.
Lock/unlock семантика: поток захватывает mutex перед критической секцией, освобождает после. Другие потоки блокируются
на lock() пока владелец не сделает unlock(). Binary semaphore по сути - два состояния (locked/unlocked). Ownership
важен - только владелец может unlock. Deadlock риск при неправильном порядке захвата нескольких mutex'ов.
