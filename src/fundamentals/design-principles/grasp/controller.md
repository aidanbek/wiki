Первый объект за UI-слоем, координирующий системные операции - фасад или use case объект. Может быть Controller всей системы (SystemController) или сценария (ProcessSaleController). Не содержит бизнес-логики, только делегирует запросы экспертам. Разделяет UI от доменной модели, точка входа для каждого действия пользователя.
