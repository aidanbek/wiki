Класс B должен создавать экземпляры A, если B агрегирует/содержит A, записывает A, тесно использует A или имеет данные
инициализации для A. Order создаёт OrderLine, ShoppingCart создаёт Order. Централизует логику создания, предотвращает
разбросанные new по всей системе. Не применяется для сложного конструирования - там нужны Factory.
