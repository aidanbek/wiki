Ответственность назначается классу, обладающему максимумом информации для её выполнения. Если объект Order содержит
список items и их цены, то метод calculateTotal() должен быть в Order, а не в каком-то OrderManager. Минимизирует
coupling, максимизирует cohesion, делает систему интуитивной - логика там, где данные. Фундаментальный принцип, из
которого вытекают остальные.
