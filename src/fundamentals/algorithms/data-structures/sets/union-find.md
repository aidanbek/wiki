Union-Find (Система непересекающихся множеств)

Структура данных поддерживающая две операции над disjoint sets: Find (определить к какому множеству принадлежит элемент)
и Union (объединить два множества). Реализация через лес деревьев где каждое дерево представляет множество, корень -
representative элемента. Find возвращает корень дерева, Union объединяет деревья делая корень одного ребёнком другого.
Наивная реализация O(n) per operation из-за вырождения в список, оптимизации критичны. Union by rank/size - присоединяй
меньшее дерево к большему, гарантирует O(log n) высоту. Path compression - при Find сжимай путь к корню делая все узлы
прямыми детьми корня. С обеими оптимизациями амортизированная сложность O(α(n)) где α - inverse Ackermann function,
практически константа. Используется в Kruskal's algorithm, динамической связности графов, image processing (connected
components).
