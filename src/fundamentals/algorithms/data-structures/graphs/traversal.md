Traversal (Обход графов)

Систематическое посещение всех вершин графа - два фундаментальных алгоритма с разными стратегиями. BFS (Breadth-First
Search) - обход в ширину через очередь: посещаем все соседи уровня k перед переходом к k+1, находит кратчайший путь по
количеству рёбер, O(V+E) сложность. DFS (Depth-First Search) - обход в глубину через стек/рекурсию: идём максимально
глубоко по одной ветке перед backtracking, находит циклы, топологическую сортировку, компоненты связности, O(V+E). BFS
для shortest paths в unweighted, level-order процессинга, DFS для структурных свойств (cycles, connectivity,
articulation points). Оба требуют marking visited вершин для избежания infinite loops в циклических графах.
