Insertion Sort (Сортировка вставками)

Построение отсортированного массива по одному элементу - берём элемент из неотсортированной части и вставляем в
правильную позицию отсортированной. Аналог сортировки карт в руке - берёшь карту, двигаешь назад пока не найдёшь место.
O(n²) worst case (reverse sorted), O(n) best case (already sorted) - adaptive алгоритм. Stable, in-place, эффективен для
малых массивов (n<10-20) и nearly sorted данных. Используется как базовый случай в quicksort/mergesort для малых
подмассивов. Online алгоритм - может сортировать по мере поступления данных. Простой, быстрый на практике для small n,
встроен в timsort как building block.
