CDN (Content Delivery Network)

Географически распределённая сеть edge серверов кэширующих статический контент близко к пользователям - изображения, CSS, JS, видео. Point of Presence (PoP) в различных регионах мира, запрос направляется к ближайшему. Снижает latency (географическая близость), разгружает origin (большинство запросов к CDN), защищает от DDoS (распределённая инфраструктура). Push (загрузка контента заранее) vs Pull (cache on first request) модели. Примеры: Cloudflare, Akamai, AWS CloudFront. Критично для глобальных приложений и медиа-контента.
