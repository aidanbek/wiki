Purge

Явное удаление конкретной записи из кэша по ключу - точечная invalidation. cache.delete(key) или PURGE /articles/123 HTTP метод. Немедленное удаление, следующий запрос промахнётся и загрузит свежие данные. Используется при обновлении/удалении сущности - user обновил профиль, purge кэша этого профиля. Требует знания точного ключа и достижения всех кэш-узлов в распределённой системе. Самая точная invalidation, но требует инфраструктуры для отслеживания зависимостей (что purge'ить при изменении).
