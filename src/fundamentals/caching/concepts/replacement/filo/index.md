Удаление самой новой записи - stack семантика, противоположность FIFO. Редко используется для кэширования, больше для специфичных структур данных (call stack, undo history). В кэшировании может быть полезна для временных данных где новое быстро устаревает. Контринтуитивна для типичных кэш-паттернов. Практически не встречается в production кэшах.
