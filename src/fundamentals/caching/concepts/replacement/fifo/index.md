Удаление самой старой записи по времени добавления в кэш - простейший алгоритм, queue семантика. Не учитывает паттерны доступа, только порядок добавления. Легко реализовать (circular buffer), минимальный overhead, предсказуемое поведение. Плохая hit rate для большинства workloads - популярная но старая запись удалится раньше недавней редкой. Используется когда доступ равномерный или кэш как temporary buffer (не true cache). Baseline для сравнения более умных политик.
