System-Level (Общее)

Кэширование на уровне инфраструктуры и сетевых компонентов - прозрачное для приложения, управляется HTTP заголовками и конфигурацией серверов. Многоуровневая иерархия: browser → CDN → reverse proxy → origin server. Каждый уровень имеет свои характеристики размера, latency, управления. Снижает нагрузку на backend, уменьшает latency для пользователей, экономит bandwidth. Требует правильной настройки cache headers (Cache-Control, ETag, Expires) для баланса между freshness и hit rate.
