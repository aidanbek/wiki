Reverse Proxy

Сервер между клиентами и origin сервером, кэширующий динамические и статические ответы - Varnish, Nginx, Squid. Терминирует клиентские соединения, агрегирует запросы к backend, кэширует результаты. Может кэшировать персонализированный контент (Vary header), применять сложные правила invalidation, compression, SSL offloading. Гибкая конфигурация через VCL (Varnish), purge API для invalidation. Первая линия защиты origin сервера, может обрабатывать 10x-100x больше нагрузки.
