Minimum Spanning Tree (Минимальное остовное дерево)

Подмножество рёбер связного взвешенного графа соединяющее все вершины с минимальной суммой весов - дерево (V-1 рёбер, acyclic, connected). Применения: network design (минимизация кабеля для соединения узлов), clustering, approximation для TSP. Kruskal's Algorithm - sort рёбра по весу, добавляй если не создаёт цикл (Union-Find для проверки): O(E log E), хорош для sparse графов. Prim's Algorithm - растущее дерево из стартовой вершины, добавляй минимальное ребро к новой вершине: O((V+E)log V) с priority queue, хорош для dense графов. Оба greedy алгоритмы, гарантированно находят optimal из-за matroid properties графов. Cut property и cycle property доказывают корректность - любое минимальное ребро crossing cut входит в MST.
