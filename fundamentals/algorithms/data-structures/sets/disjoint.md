Disjoint Sets (Непересекающиеся множества)

Коллекция непересекающихся подмножеств элементов - partition универсального множества на взаимоисключающие группы.
Каждый элемент принадлежит ровно одному множеству, объединение всех множеств даёт исходное универсальное. Используется
для представления equivalence relations - элементы в одном множестве эквивалентны друг другу. Классические применения:
компоненты связности в графах (вершины в одной компоненте - одно множество), кластеризация, Kruskal's MST алгоритм (
проверка создания цикла). Эффективная реализация через Union-Find структуру позволяет быстро объединять множества и
проверять принадлежность одному множеству. Динамическая структура - можно добавлять элементы и объединять множества, но
обычно нельзя разделять.
