Searching (Общее)

Фундаментальная операция нахождения элемента в коллекции по критерию - основа информационных систем, баз данных, поисковых движков. Два основных измерения: критерий поиска (точное совпадение, диапазон, паттерн) и структура данных (массив, дерево, граф, хеш-таблица). Efficiency зависит от организации данных - unsorted требует перебора, sorted позволяет оптимизацию, indexed даёт O(1) в лучшем случае. Trade-off между временем поиска и overhead поддержки структуры (sorting, indexing). Оптимизации: pre-processing (sort, build index), approximate search (bloom filters), parallel search, early termination.
