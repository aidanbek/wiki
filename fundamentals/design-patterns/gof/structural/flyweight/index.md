Разделение состояния объектов для эффективного использования памяти при большом количестве мелких объектов. Intrinsic state (shared) vs extrinsic state (unique) - шрифт объекта Character один для всех, позиция уникальна. Factory управляет пулом flyweight'ов, переиспользуя объекты с одинаковым intrinsic state. Критично для рендеринга текста, игровых объектов, графических систем. Производительность и память за счёт некоторой сложности.
