Гарантирует единственный экземпляр класса и глобальную точку доступа к нему. Private constructor, static getInstance() возвращает shared instance. Решает координацию действий через единую точку - логгер, конфигурация, connection pool. Считается антипаттерном - глобальное состояние, проблемы тестирования, скрытые зависимости. Используй DI контейнеры для управления lifecycle вместо ручного Singleton. Допустим для stateless services и инфраструктурных объектов.
